# ComfyUI-LTX-Prompt-TY (Pure Local) 🎬

专为 **LTX-Video 2.3** 打造的 100% 本地反推提示词导演节点。

## 🛠️ 更新亮点
- **纯净本地**：移除了云端 API 逻辑，仅支持 Ollama。
- **最新模型**：默认 **qwen3.5:9b**，支持 Abliterated 及 qwen3-vl 视觉模型。
- **对话强对齐**：通过内部协议确保台词不串角色。

## ⚙️ 环境
- **Ollama**: 请确保本地已安装并在后台运行。
- **Python**: 3.12.10
- **PyTorch**: 2.8.0
