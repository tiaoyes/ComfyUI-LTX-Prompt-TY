# ComfyUI-LTX-Prompt-TY (Ultimate Director Edition) 🎬

专为 **LTX-Video 2.3** 优化的 100% 本地、逻辑补全型提示词导演节点。

## 🌟 核心升级内容
1. **角色/对白强绑定**：修正多图环境下对白归属错误的缺陷。
2. **多帧衔接逻辑**：分析多图间的位移与构图变化，生成连续的运镜轨迹。
3. **术语升维转换**：自动将简易剧本指令（如"靠近"）转化为专业摄影术语（`Dolly-in`）。
4. **官方七步法结构**：严格遵循 LTX 2.3 官方博文推荐的 Prompt 逻辑结构。
5. **显存回收策略**：支持推理后立即卸载模型，确保显存优先供给 LTX 采样器。

## 🛠️ 环境要求
- Python 3.12.10
- PyTorch 2.8.0
- Ollama 本地服务

## 🤖 支持的模型
| 模型 | 说明 |
|------|------|
| qwen3-vl:4b | 默认模型，视觉语言模型 |
| qwen3-vl:8b | 视觉语言模型（更大参数） |
| qwen3.5:4b | 纯文本模型 |
| qwen3.5:9b | 纯文本模型（推荐） |
| qwen3.5:27b | 纯文本模型（最大参数） |
| qwen3.5-abliterated:4b | 无审查版本 |
| qwen3.5-abliterated:9b | 无审查版本 |
| qwen3.5-abliterated:27b | 无审查版本 |

## 📝 输入参数说明
| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| 图片_1 ~ 图片_10 | IMAGE | - | 最多支持10张参考图片 |
| Ollama地址 | STRING | http://localhost:11434 | Ollama 服务地址 |
| 模型选择 | 下拉菜单 | qwen3-vl:4b | 选择本地 Ollama 模型 |
| 显存策略 | 下拉菜单 | 立即卸载 | 模型显存驻留策略 |
| 角色与剧本分镜 | 多行文本 | 见下方示例 | 角色特征与剧本内容合并输入 |
| 对话语言 | 下拉菜单 | 对话保留中文 | 输出语言策略 |
| 随机种子 | INT | 0 | 控制生成一致性 |

### 角色与剧本分镜 示例
```
【角色】跳爷: 银发, 深蓝唐装. 徒弟: 黑色汗衫.
【剧本】跳爷对徒弟说：'这就是真功夫。'
```

## 💡 显存策略说明
- **立即卸载**：推理完成后立即释放显存（推荐显存紧张时使用）
- **5分钟驻留**：模型在显存中保留5分钟，适合连续生成
- **始终驻留**：模型常驻显存，适合批量任务
